# Adventa CNC Solutions Website

This repository contains the source code for the official Adventa CNC Solutions website, [adventacnc.com.tr](https://adventacnc.com.tr). It is a modern, performant, and multi-language static site built with Jekyll.

## Live Demo

The production site is available at: **[https://adventacnc.com.tr](https://adventacnc.com.tr)**

## Core Technologies

*   **Jekyll**: A static site generator that transforms plain text into a complete website.
*   **Liquid**: A flexible templating language used to dynamically build pages and insert content.
*   **Sass/CSS**: For styling and responsive design.
*   **YAML**: Used for managing all site content, including translations and product specifications, ensuring a clean separation of data from presentation.

## Running Locally

To run the site on your local machine, you will need Ruby and Bundler installed. For detailed, OS-specific instructions, please follow the official [Jekyll Installation Guide](https://jekyllrb.com/docs/installation/).

Once your environment is set up, follow these steps:

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/your-username/adventacnc.git
    cd adventacnc
    ```

2.  **Install dependencies:**
    ```sh
    bundle install
    ```

3.  **Run the Jekyll server:**
    ```sh
    bundle exec jekyll serve
    ```

4.  **Open your browser** and navigate to `http://localhost:4000`. The site will automatically reload whenever you save a change to a file.
